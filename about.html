About Me

Hi there! I am Nikola, a passionate soul that loves developing mobile apps.
I also enjoy sports, working out and much more.

How It Started

My first experience with programming was not that long ago. I decided to try learning Python. It was fun building a simple fitness calculator using Tkinter (standard GUI library for Python). I also developed a simple website bot that uses Selenium.

It occurred to me: I just built a simple app! So I wanted some more. Soon after, I realized that I have an iMac and that I could get into iOS development. After watching a few videos about iOS development, I was sure that I wanted to do this as a profession.

How Is It Going

Since then, I have been learning everything iOS related. So far, I built one app using SwiftUI but, there will be many more to come. I also started writing articles about iOS development to get a better understanding of some of the concepts.

Skills

Skills in iOS development and design empower me to deliver high-quality apps.

  • Proficient in writing Swift and building apps with Xcode.
  • Well-versed with many Apple frameworks like SwiftUI, CoreData, WidgetKit, WatchKit, and other third-party libraries.
  • Experience working with Firebase and communicating with REST APIs.
  • Comfortable building UI using SwiftUI.
  • Worked with Figma and GIMP to handoff and implement designs.
  • Used Git and GitHub for version control and to share my code through open source.
